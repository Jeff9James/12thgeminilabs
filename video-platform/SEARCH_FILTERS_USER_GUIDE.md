## Quick Start

After performing a search, you'll see new **Sort** and **Filter** controls above your results.

## How to Sort Results üîÑ

### Using the Sort Dropdown
Located at the top-left of your search results:

1. Click the **Sort dropdown** (shows "Sort by Relevance" by default)
2. Select your preferred sorting option:
   - **Sort by Relevance** - Best matches first (default)
   - **Recently Uploaded (Newest First)** - Latest uploads at top
   - **Recently Uploaded (Oldest First)** - Earliest uploads at top
   - **Recently Used (Newest First)** - Files you interacted with recently
   - **Recently Used (Oldest First)** - Files you haven't used in a while
   - **Name (A-Z)** - Alphabetical order
   - **Name (Z-A)** - Reverse alphabetical order

Results update instantly when you change the sort option.

### What is "Recently Used"?
Files are marked as "used" when you:
- Chat with them
- Analyze them

This helps you quickly find files you've been working with.

## How to Filter Results üéØ

### Opening the Filter Panel

1. Look for the **Filter** button next to the sort dropdown
2. Click it to reveal the filter options
3. Click again to hide the panel

**Tip**: The button turns blue when filters are active or panel is open.

### Filter by File Type

#### Include Only Specific Types
Use this when you want to see ONLY certain file types:

1. Find the "**Include Only:**" section under "File Types"
2. Click on file type buttons you want to see:
   - Video
   - Audio
   - Image
   - PDF
   - Document
   - Spreadsheet
   - Text
3. Selected types turn **green**
4. Results now show ONLY those types

**Example**: Click "Video" and "Image" to see only video and image results.

#### Exclude Specific Types
Use this when you want to hide certain file types:

1. Find the "**Exclude:**" section under "File Types"
2. Click on file type buttons you want to hide
3. Selected types turn **red**
4. Results no longer show those types

**Example**: Click "PDF" to hide all PDF files from results.

### Filter by Specific Files

#### Include Only Specific Files
Use this when you want to search within specific files:

1. Find the "**Include Only:**" section under "Specific Files"
2. Scroll through the list of your uploaded files
3. Check the boxes next to files you want to search
4. Results show ONLY matches from those files

**Example**: Select "Meeting Recording.mp4" and "Project Notes.pdf" to search only within those two files.

#### Exclude Specific Files
Use this when you want to ignore certain files:

1. Find the "**Exclude:**" section under "Specific Files"
2. Check the boxes next to files you want to exclude
3. Results no longer show matches from those files

**Example**: Uncheck "Old Video.mp4" to remove it from search results.

### Understanding the Results Counter

At the top-right, you'll see: **"Showing X of Y results"**
- **X** = Number of results after filtering
- **Y** = Total results before filtering

This helps you see how many results were filtered out.

### Clearing Filters

When any filters are active, a **"Clear Filters"** button appears:
1. Click the red **"Clear Filters"** button
2. All filters are instantly removed
3. You see all search results again

## Tips & Tricks üí°

### Combining Filters
You can use multiple filters together:
- Include certain types AND exclude specific files
- Sort by name AND filter by type
- Exclude types AND exclude specific files

### Filter Badge
The blue badge on the **Filter** button shows the total number of active filters. This helps you quickly see if filters are applied.

### Performance
All filtering happens instantly in your browser - no need to search again when applying filters!

### Mobile Usage
On smaller screens, the filter panel stacks vertically for easier access.

## Troubleshooting ‚ùì

### "No results found" after filtering
- Check if filters are too restrictive
- Click "Clear Filters" to reset
- Try using "Include" OR "Exclude", not both

### Can't find a file in the filter list
- File might not be uploaded yet
- Check "My Files" page to verify upload
- Refresh the search page

### Sort not working as expected
- "Recently Used" requires interaction history
- Files without usage dates appear at bottom
- "Recently Uploaded" requires upload dates

### Filter button won't open
- Try refreshing the page
- Ensure JavaScript is enabled
- Check browser console for errors
